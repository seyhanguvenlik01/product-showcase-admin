## Introduction

Product Showcase Admin is a web-based application for managing and showcasing products effectively. This tool allows administrators to manage product listings, view statistics, and handle customer inquiries.

## Features
- User authentication and authorization
- Product management (add, edit, delete)
- Showcase of products on frontend
- Analytics dashboard
- Easy-to-use interface

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/seyhanguvenlik01/product-showcase-admin.git
   ```
2. Navigate to the project directory:
   ```bash
   cd product-showcase-admin
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server, run:
```bash
npm start
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to see the application in action.

## API Endpoints
| Method | Endpoint             | Description                 |
|--------|----------------------|-----------------------------|
| GET    | /api/products        | Get all products            |
| POST   | /api/products        | Create a new product        |
| PUT    | /api/products/:id    | Update product details      |
| DELETE | /api/products/:id    | Delete a product            |
